
#  Warehouse Management System - Frontend

## Overview
React.js frontend application untuk sistem manajemen gudang dengan role-based authentication, responsive design, dan CRUD operations.

## Tech Stack
- **Framework**: React.js 18+
- **Routing**: React Router DOM v6
- **HTTP Client**: Axios dengan interceptors
- **UI Library**: React Bootstrap 5 + Bootstrap CSS
- **Notifications**: React Toastify
- **State Management**: React Hooks (useState, useEffect)
- **Testing**: React Testing Library + Jest
- **Build Tool**: Create React App

## Quick Start

### Prerequisites
- Node.js 18+
- Backend API berjalan (port 5000)

### 2. Installation
```bash
# Clone repository
git clone <repo-url>

# Install dependencies
npm install

# Run
npm start
